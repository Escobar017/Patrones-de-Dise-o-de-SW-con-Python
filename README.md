# Patrones-de-Diseño-de-SW-con-Python
Curso Patrones de Diseño de Software en Python - Curso Práctico

A) Los Patrones de Diseño.

Los Patrones de Diseño, en sencia son soluciones bien establecidas y probadas a problemas comunes en el diseño de Software

Se pueden pensar en ellas como platillas que uno puede seguir para abordar ciertos problemas que se presentan con frecuencia.

Pero ojo esto no quiere decir que sean fragmentos de codigo que uno simplemente pueda copiar y pegar, son más bien guias que puedes
seguir y que tendremos que adeptar a nuestro problema en especifico.

Los patrones de diseño ayudan a:

1. Acelerar el proceso de desarrollado
2. Ayudan hacer el codigo mas legible y reutilisable
3. Facilitan la comunicacion entre desarrolladores

Hay 3 tipos de Patrones de diseño

Creacionales - Se ocupan en la creación de objetos
Estrucrurales - Tratan la composicion de clases y objetos
Comportamiento - Se centran en la comunicación entre diferentes objetos en un sistema

Los Patrones de diseño, son tecnicas que de programación que se centran en resolver problemas recurrentes a nivel de implementación,y diseño de software

Los patrones de diseño, en general son aplicables in impñortar la arquitectura de software que se este usando.

-- ¿Qué es una clase?

Una clase es como un molde o una plantilla que define cómo será algo dentro de un programa.

La clase indica:

qué características tendrá ese algo (atributos)
qué acciones podrá hacer (métodos)
Ejemplo:

Imagine la clase Auto.

La clase puede definir que todo auto tiene:

color
marca
modelo

Y también puede hacer cosas como:

arrancar()
frenar()
acelerar()

Entonces, la clase no es el auto real, sino la idea o estructura de cómo debe ser.

-- ¿Qué es un objeto?

Un objeto es una instancia real de una clase.

Es decir:

la clase = el molde
el objeto = el elemento creado a partir de ese molde
Ejemplo:

Si la clase es Auto, entonces los objetos pueden ser:

un auto rojo Toyota Corolla
un auto negro Honda Civic
un auto azul Nissan Versa

Todos son objetos de la clase Auto, pero cada uno tiene valores diferentes.
